# 稽查案件追蹤

手機優先的私人稽查案件與現場處理紀錄系統。

## 功能

- 儲存水號、姓名、電話、地址、座標、表號、案件原因與狀態
- 分別上傳 3-1 畫面及圖資畫面
- 同一案件可持續新增多次電話、現勘或內部處理紀錄
- 手機直接拍照、錄影或選擇多個檔案上傳
- 搜尋水號、姓名、地址、表號及案件原因
- 顯示下次追蹤日期、今日追蹤及逾期案件
- 只有網站擁有者登入後才能查看資料

## 資料安全

GitHub 只保存網站程式碼。案件資料保存在受登入保護的資料庫；照片與影片保存在私有檔案空間，下載時也會再次驗證登入者。

## 儲存結構

- D1：案件、處理紀錄、檔案索引
- R2：3-1 畫面、圖資畫面、現場照片及影片
- Sites access policy：僅擁有者帳號可開啟
